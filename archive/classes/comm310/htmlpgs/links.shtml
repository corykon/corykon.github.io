Hyperlinks

Hyperlinks are really what make the internet so special. They allow you to click and then be transported to a new page. Without them, websites would just be big online books, we'd have to flip or scroll through until we found what we were looking for. On this page I'll explain the basic of creating hyperlinks with HTML.

Hyperlinks are created using the <a> tag, which stands for "anchor". Within this html tag there are two attributes which should be included, the href or "hyperlink reference" and the title attribute. The href attribute is used to specify the location where the link should take you, and the title attributes basically explains where the link takes you. The <a> will be wrapped around the text or image you would like to serve as the link, in this manner:



<a href="/path/path.html" title="Link Title">This text will be a link. </a>

Again, the href specifies the path this link will follow, whether that be to another page, a location within the same page, or to a particular file. The title attribute explain the link. As you hover over a link, the title will be displayed, sort of like a tooltip.

Paths in in the href attribute can be absolute or relative. Absolute paths begin with the "/" symbol, which means "go back to the root of this webpage's file structure. Without that symbol, the path is assumed as being relative to the location of the file you are currently on.
