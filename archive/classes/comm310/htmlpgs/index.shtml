HTML (Hyper Text Markup Language)

HTML is where writing code for the internet begins. It is used to create the basic structure of a webpage- not from a presentation standpoint, but from the content/functionality side of things. Hyper essentially means "linkable." HTML is what enables one page to link to another. Markup refers to HTML's ability to apply meaning to text, you can use it to tell the browser that certain text is a title, or that it is more important than other text, or any other number of things.

Content Vs. Presentation

This is one of the most important principles taught in the class– HTML is primarily used for giving a website content and structure, while CSS is used for the presentation and appearance of the webpage. The HTML Dog textbook stresses this principle early on, probably in a much clearer manner than I can, so to quote a portion of its explanation:

"HTML is for one thing and CSS is for the other– HTML should be used solely to construct content and CSS should be solely to present it."

The point was further illustrated in this manner:

HTML = Content

CSS = Presentation
