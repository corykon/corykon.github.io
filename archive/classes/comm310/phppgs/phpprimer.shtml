Intro

So, what is PHP? It's an open-source scripting language. Open source means that the materials used to create the language are completely free and open to the public, and scripting means it's a language that does not need to be precompiled to work; the code is just processed on the fly as necessary.

What does PHP do? It's used for the dynamic aspects of your website, the things that allow the user to really interact with the page. It can grab information that the user inputs into forms and use it in lots of various ways, it can change the look of a page based on the person viewing it, or even just based on the browser being used. Things such as blogs or discussions threads often use PHP to store and display user comments on a page.

I think it is also important to note that PHP code is executed by a server, not by your browser. For this reason, when you are testing a PHP page on your local machine, you'll end up with a blank page if you try simply loading it up on a browser. You must be running a server on your computer, such as Apache, in order for the PHP statements to be executed. Most all hosts have some type of server running to interpret PHP

How to Code

PHP Basics (code block, comments)

We'll start with how to begin and close a block of PHP code, and how to add comment to the code. Similar to html and javascript, PHP needs to be contained within an opening and closing "tag". Also comments can be used, which will be ignored by the server executing the code. Here is what this looks like:


<?php

    // This is a single line php comment

    #  This is also a single line comment

    /* This is a multiple

    *  line comment

    */
>

Remember: The opening and closing PHP "tags" must be exactly as displayed above, otherwise it will not be recognized as PHP. Also, be sure that the multi-line comment is closed, otherwise all the code located after the comment will be ignored.

Variables

A variable is used to hold a value, kind of like a storage container, whether it be numbers, text, or boolean values. In many programming languages a variable type must be declared, but in PHP this is not necessary. Here is an example of the syntax for variables:


<?php

    $var = 'This is a variable';

    $another_var = 'This is another variable';

    echo '$var';

    echo '$another_var';

>

Remember: When using variables, it is important to always use the $ sign before the variable name. Variable names can include numbers and the underscore characters, but a letter has got to be the first character. Also, remember that PHP variable ARE case sensitive, so $VAR is not the same as $var.

Strings - echo, print, single vs. double quotes

As was shown in the previous example, the echo command is one way to display strings onto the screen in PHP. The other way is to use the print command. Both commands use the exact same syntax. Here is an example of this:


<?php

    $var = 'Optimus Prime';

    echo 'My name is $var';

    echo "My name is $var";

>

Remember: It is possible to use both single and double quotes with the print and echo statement, and they yield different results. If you use double quotes, any variable name included will be expanded. In other words, the value contained in the variable, instead of the variable name, will be printed. If you use single quotes then the opposite is true, the variable name will be displayed on the screen.

Arrays (Numeric)

An array is simply a variable which can hold more than one value. It simplifies things when we need to hold a large number of values, because instead having to create a massive number of variables, we create just one that can hold a massive amount of values. The code to create arrays can be done a couple different ways:


<?php

    //like this:

    $classes = array ("CIT 320", "CIT 336", "CIT 330");

    //or like this:

    $classes2[] = "CIT 320";

    $classes2[] = "CIT 336";

    $classes2[] = "CIT 330";

    //here is how to reference a particular value in the array:

    echo "$classes[1];

>

Remember: When referencing a value in the array, the first location value will by default be 0. So if you would like to reference the third value you entered into the array, you would call for the value at location "2", not location "3".

Arrays (Associative)

In an associative array, values are referenced by a name rather than by a numeric number. These names are specified by you, and can be a huge help in remembering which values are stored in which locations. Here are a couple examples of how this can work:


<?php

    //like this:

    $meals = array(

        morning => "breakfast",

        afternoon=> "lunch",

        evening=> "dinner"
    );


    //or like this:

    $meals2[morning] = "breakfast";

    $meals2[afternoon] = "lunch";

    $meals2[evening] = "dinner";

    //here is how to reference a particular value in the array:

    echo "$classes[morning];

>

Remember: The rules mentioned for single and double quotes still apply here. Single quotes will create "literal" strings, and double quotes will retrieve variable values.

Arrays (Multidimensional)

Multidimensional arrays are basically arrays within arrays. You could think of an array as a table with one row, and a multidimensional array as table with multiple rows. Here is an example of the syntax for creating a multidimensional array:


<?php

    $food = array (

	        array ( name => "apple",

	                type => "fruit",

	                color => "red"),

	        array ( name => "carrot",

	                type => "vegetable",

	                color => "orange")
                      );
>

Remember:When referencing a value in a multidimensional array, you first refer to the "array within the array" by number, and then to the specific location in that array. (Like this "$food[1][name];")

Operators

Operators are used to perform operations on numbers, strings, and or variables such as arithmetic, assignment, or combining strings together. The operators in PHP are:

Assign "="
Add"+"
Subtract"-"
Multiply"*"
Divide"/"
Append(to combine strings)"."

An example of code using these operators:


<?php

   $var = 5 + 2 - 4 * 4 / 3;

   $var2 = "The answer is: ". $var;

    echo $var2;

>

Remember: Arithmetic in PHP will follow the normal order of operations, i.e. multiplication and division before addition and subtraction.

Conditional Statements

Conditional Statements are what truly start to make web pages dynamic. They enable different pieces of code to be executed, based on things like user input, or possibly the time, or the type of browser currently being used. The if statement evaluates a variable or expression, and if it returns true, then a particular piece of code is executed. There also exists the else statement, which contains code to be executed if the expression returns false. Additionally, there is the else if statement, so you can run multiple comparisons. The switch statement provides the same type of functionality, but the syntax is optimized for a larger number of comparisons. Following are examples of the if and switch statements:


<?php

   if ($var = 1){

        echo '$var equals 1';

   }else if ($var = 2){

        echo '$var equals 2';

   }else {

        echo '$var does not equal 1 or 2';

   }

   //Similar code could be written in this way with the switch statement:

    switch ($var) {

        case 1:

            echo '$var equals 1';

            break;

        case 2:

            echo '$var equals 2';

            break;

        default:

            echo '$var does not equal 1 or 2';

            break;
    }
>

Remember: The {} brackets need to surround the code that is dependent on a conditional statement while using the if, but they aren't needed with the switch.

Loops

Loops can be an especially handy tool, because they allow the same piece of code to be rerun up to an infinite number of times, and can also act as "counters." Loops work using some type of conditional statement; as long as the statement returns true, the code within the loop will continue to be executed. There are a few types of loops, including for, do, do while, and foreach. Following are simple examples to show their syntax. (NOTE: the following code would not be fully functional, it is only to gives examples of syntax.)


<?php


    //while loop

    while ($var < 10){

        $var += 1;

        echo "$var";
    }



    //do while loop

    do{

        $var += 1;

        echo "$var";

    }while($var < 10);


    //for loop

    for($var = 1, $var < 10, $var++){

        $var += 1;

        echo "$var";

    }

    //for each loop (iterates through an array)

    $avar = array(1,2,3,17);

    foreach($avar as $v){

        echo "$v";

    }

>

Remember: Each of these loop types is useful in certain situations, evaluate each before deciding on which to use. Also take notice that variables used in while loops (as well as arrays in foreach loops) must be declared beforehand,but they CANNOT be declared before a for loop is used.
Source of "foreach" syntax: php.net- "Foreach"
