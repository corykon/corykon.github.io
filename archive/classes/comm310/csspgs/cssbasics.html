CSS Basics

Although CSS works hand and hand with the HTML of a web page, its syntax is completely different. It is made up three major parts, namely selectors, properties, and values. These three together are what is called a CSS rule. The following would be an example of a standard CSS rule, with one selector, property, and value: 


p {text-size: 1em;}

It the preceding example, p is the selector. This tells the browser that the <p> in the linked HTML page is what is being targeted for some styling. The text-size: portion of this rule is the property. The particular property as you can probably guess gives you the ability to change the zie of the text in that HTML paragraph. Finally, 1em is the value. (An em is a relative typography measurement). The text in the paragraph tags of the linked HTML page will be displayed at a size of 1 em.

Make a special note of the colon placed between the property and value. This must be present for the CSS to work. An equals sign, or any other symbol will not work. Also, remember not to forget the semicolon which follows the value. In a singe property rule it will not cause problems, but if there are multiple properties in the rule, the browser will not properly interpret your CSS.
