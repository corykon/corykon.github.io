Styling Links with CSS

Hyperlinks can be style by the same css properties explained in the text section, but there are a few points specific to hyperlinks that I'd like to explain here. First of all, you need to be aware that the css styling of font-color applied to a paragraph will not apply to the hyperlinks within that paragraph, because by default hyperlinks are blue. Similarly, the default underlining can be removed with css applied to the paragraph tag. CSS for hyperlinks needs to be applied to the <a> tag itself.

The most important concept to learn when styling hyperlinks is that they can exist in four different states. Here they are with their associated psuedo classes, which we'll discuss in a minute:

  • Normal - a:link
  • Visited - a:visited
  • Hover - a:hover
  • Active - a:active

A visited hyperlink is one that references a page the user has already visited, whether by clicking on that particular hyperlink, or visiting it by another means. By default, visited links are that nasty purple color. A hyperlink is in its hover state when the mouse roll's over it. Keeping that in mind, we can create cool rollover effets, like link colors changing or underlines appearing when a user rolls over your links. The active state is when a link is currently being clicked (or when browser considers the link "in focus", so this state usually doesn't exist for long. Normal links are whenever the link isn't in any of the other three states. As mentioned, the default for a normal link is the blue color we're all used to seeing.

After understanding the different hyperlink states, we can use CSS psuedo classes to create cool effects. Simply choose a style for normal links, apply that style using the a:link selector(or just a), and then decide how you want the link to look when its being hover over, and apply that style with the a:hover selector. I typically just apply the normal styling to the a rather than a:link, because apply style to a affects all link states, including the visited and active states. If you want those states different as well (which would be a good idea for the visited state, at least when the hyperlink is in the body of the text as opposed to the main navigation), then go ahead and do so.

Here's an example of this in action. This is the css used to style the links in the body of this webpage:



a{
	color: #0099cc;
	text-decoration:underline;
}


a:hover{
	color: #0080A6;
}


a:visited {
    color: #6dc7ec;

}
