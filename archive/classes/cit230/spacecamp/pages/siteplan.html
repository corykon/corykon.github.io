Site Plan



Purpose

Simply put, the purpose of this website is to inform parents and potential participants about space camp. That purpose can be divided into two main parts:

  • Attracting customers, getting them excited about space camp
  • Informing parents, helping them feel confidence in our staff and facility

Audience

The audience of this website would be children interested in attending Space Camp, and also their parents. This site needs to showcase the quality of activities and the fun that will be had, as well as the highly quality staff and facilities to inspire confidence in the minds of the parents.

Why will people use this website?

People will use this website if they would like to attend a summer camp and are searching for one to attend. Also, some will have already signed up to attend but just want specific information about the camp.



Basic Site Components


Directories

The following directories will be included:

  • css - for css files
  • images - for all images
  • modules- for nav, footer, and header modules
  • pages - for all pages besides the index
  • activities-(a sub-directory of pages) contains activity pages

Modules

Header
The header module will contain the logo and tagline, as well as a search box in the upper right hand side.

Navigation
This module will contain the five main links. They will be displayed over a metalic-textured background to give a high tech feel.

Activities Navigation
Secondary navigation will be diplayed to the left of the content on each of the activities pages.

Footer
The footer wil contain copyright info, and will contain links to the colophon, contact page, site plan, resources, and copyright page.

Pages

Home Page
This will contain a welcome, and a brief intro to the purpose of the website. A large picture of a child in an astronaut suit will be prominently featured.

Activities
This page will contain a brief general intro about the activities at our space camp, and links to pages about each activity.

Qualifications
This page will outline the qualifications that a child must meet to be able to participate.

Staff
This page will introduce our staff, and show their experience and credentials.

Contact
This page will aid potential particpants in contacting us and asking questions about the camp.

Style Guide



Layout

This website will be on a centered template that is 900 pixels wide. The header is 100 pixels tall, and the navigation bar is 40 pixels tall. Their is a left and right section in the center. The right section is for picture/secondary nav and is always a smaller portion(except on homepage). The footer is 60 pixels tall.

Browser Requirements

All pages should be viewable in the major browsers: Internet Explorer, Mozilla Firefox, Opera, and Google Chrome w/ minor if any differences.

Graphics

Images included should be one of the common suportable images types, such as jpeg, png, and gif. They should be appropriately sized, and geared towards optimum loading speed. A large amount of images should no be displayed on a single page.

Colors

The color scheme is composed, black/spacey colors and orange. The orange colors hex code is #F26522. A standard white background will be used in the content section of the page, and purple/black space background will be used.

Typography

Body Text

  • Font Family defined as "Georgia, 'Times New Roman', serif;"
  • Font size: 12px
  • Font Color: #292421;

Heading/Subheadings

  • Font Family defined as "Georgia, 'Times New Roman', serif;"
  • H4-H6- Font Color: #292421;
  • H1-H3- Font Color: #F26522;

Footer Text

  • Font Family defined as "Georgia, 'Times New Roman', serif;
  • Font size: 12px
  • Color: #FFFFFF
  • Hover Color: #F26522;

Navigation

The main navigation will be located just below the header. They will be created in the same sans-serif font as the logo using text images, and have a css dotted bottom border. When rolled over, the bottom border will turn orange (#F26522). The current page will be underlined in orange. The blue "image link" border should be taken out using css.

Links within the content of the page will be the same color as normal text, but will be underlined to show that they are links. They will change to the hex color code #F26522 when the mouse hovers over them. There is also navigation located in the footer and it is set to align with the right edge of the page. They also employ the css rollover effect.

Overall, navigation should be simple and intuitive. Links should be clearly labeled.

Content

The content of the page overall should be useful, proffesional, and concise. It should be arranged in an aesthetically pleasing and logical way. Its goal is to inspire excitement in potential camp attendees and confidence in the parents.

Logo

The logo (as seen in header)is named spacecamplogo.png. It should not be altered. It should appear prominently on every page. Use its design and the tagline as a model for other graphics created.
