The purpose of this page is to describe the efforts that have been made to insure that this website is as simple and user friendly as possible, as well as search enigine optimized. Also discussed are the efforts that have been made to insure a large ROI.


Web Standards

This webpage follows the current standards for XHTML and CSS code. This ensures that it will be viewable on all browsers,and on many different mediums, including computers, cell phones, and other devices.

Usability

Great efforts were made to ensure that this webpage is very simple and easy to use. The following lists some of the specific efforts to enhance navigation and clarity of content:

  • The names of links are simple and descriptive
  • All links employ mousever effects, such as font changing colors or appearing arrows
  • Links within the content section of the page are underlined
  • The main navigation is just below the header, and secondary navigation is located on the left, common web standards
  • Navigation is persistent, meaning it is the same on every page
  • In the main navigation, the current pages link is underlined
  • The name of each page is clearly displayed
  • The names of pages correspond with their link
  • Headings and titles are properly sized to ensure clear hierarchy and scanability
  • The design is stylish, but not overpowering. It doesn't cloud the content.
  • The purpose of each page is clearly defined
  • Content is clear and concise

Search Engine Optimization(SEO)

SEO is ensured because of a clear hierarchy, "alt" and "title" being included in each link and image to increase the natural occurence of keywords.

Return On Investment(ROI)

While great efforts were made to ensure the usability, SEO, and strict adherence to web standards, relatively little money has been spent to do so.
