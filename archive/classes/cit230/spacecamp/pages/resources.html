Resources

This page is documentation of all sources used to create this website for my CIT 230 final exam.

Programs Used

  • Adobe Photoshop- To edit images and textures.
  • Adobe InDesign- To brainstorm layout and create images for header and navigation.
  • Netbeans- To write and edit code.
  • Filezilla- To upload files.
  • Picnik- To edit pictures.
