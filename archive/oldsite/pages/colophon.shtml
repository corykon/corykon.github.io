Colophon

The purpose of this page is to describe the efforts that have been made to insure that this website is as simple and user friendly as possible.


Navigation

The navigation in this webpage is very easy to identify and use, because it follows many common web standards:

  • The names of links are simple and descriptive
  • All Links employ mousever effects, such as font changing colors or appearing arrows
  • Links within the content section of the page are underlined
  • The main navigation is just below the header, another common web standard
  • Navigation is persistent, meaning it is the same on every page
  • The main navigation has arrows indicating the current location

Headings/Titles

The name of each page is clearly displayed. It is located in the status bar, and in a large heading at the top left of the content section, which gives the user confidence as to which page they are currently on, and consequently helps with navigation. The headings of main pages match the links that correspond with them, further clarifying the navigation. Properly sized headings are used within the content in order to provide clearer visual hierarchy and greater organization, allowing for more "scanable" pages.

Simplicity

Overall this webpage has a very simple and clean design. There aren't a multitude of links, advertisments, pictures or clumps of other content pulling for your attention. You can very quickly scan the contents of the page without getting lost, and thereby quickly find what you're looking for.

Consistency

As with the navigation, elements of this webpage are consistent. The fonts, the logo, the colorscheme and layout remain the same throughout the website so that all the pages feel connected and unified. This is a very simple point, but without consistency, users feel like they are going to a completely different website each time they click on a link.
