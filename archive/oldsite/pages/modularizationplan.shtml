Modularization Plan

This plan contains descriptions of the 3 modules that will be used to display identical content on multiple pages.


Header Module

The header will be at the top of each page and span 760 pixels. It will contain the logo of my webpage, and the text "Cory Fugate - cit specialist, web and graphics designer. Just below that will be a zoomed in image of my guitar.

Navigation Module

The navigation module will be just below the header and will contain links to all the other main pages, displayed horizontally. It will be created using graphics with rollover effects. The Links will include Home, About Me, Resume, Portfolio, Interests, Assignments and a Contact link.

Footer Module

The footer will be at the bottom of the page, and will give a graphically sound end to the page. In smaller text on the left will be copyright information, and the right will be additonal links including the style guide, site plan, modularization plan, colophon, contact, and a facebook icon.
