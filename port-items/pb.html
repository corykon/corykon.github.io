Population Builder
Dev & UX. 2017-2026.

I was the UX designer and lead front end engineer for the Population Builder web application. It features a slick drag and drop interface that enables non-technical users to write complex healthcare database queries that are translated into SQL. It was a major part of an initiative for "self-service anayltics", and it was all about getting data to the masses.

In 2020, the application was awarded Microsoft's Health Innovation Award in the "Improve Operational Outcomes" category.
