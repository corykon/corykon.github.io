Cashmere
Component Library & Style Guide. 2017-2025

Cashmere is an open source design system used by healthcare tech company Health Catalyst.

We created it in 2017, and have grown it from a humble style guide with a few angular components into fully fledged design system that supports dozens of applications and projects.
