Street Skate
DOS Game. 2003.

This was made way back in high school during my very first foray into to programming. We were taught in QBasic, which runs in DOS. I love skateboarding, was a huge THPS fan, and so this was the obvious choice for my final project in that class.

I recently spent some time playing around getting this code to run on my macbook (it's a miracle I still had the code laying around on an old hard drive). Got such a kick out of seeing it again that I wanted to post it here.
